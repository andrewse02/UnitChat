Unit Chat

About

We are a simple, sleek, chatting service. Interact with your friends, publicly or in private. Explore all the things you can do with Unit Chat!

This project was created to present as a capstone for Devmountain to display the skills learned while enrolled.

Unit Chat is open-source and was built on various technologies. Such as: Node.js, Express, Socket.IO, JsonWebToken, UnderageValidate and many more.

Features

  • Send and receive messages in real-time

    You'll know when a user is typing, and you'll receive their message at almost the instant they send it


  • Edit or delete messages

    Simply hover over a message that you sent, and click the edit or delete buttons


  • Start a private conversation

    Click the new conversation icon, type the username, and start chatting
